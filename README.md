## ğŸ”„ How It Works
1. User enters a message in the chat input
2. Message is stored in Streamlit session state
3. Entire conversation history is sent to OpenAI
4. OpenAI generates a response using the selected model
5. Response is streamed live to the UI
6. Assistant reply is added back to the session

This ensures **context retention** across the conversation.
